# 个人博客网站：

##### 目标：一个简单易用，用于展示自己擅长的技能，和兴趣爱好，以及基本信息的网站

##### 调研发现：

1. 市面上的博客都比较简单易用，页面简陋
2. 我需要自己能够随时修改我博客的个人信息
3. 功能较为全面，与访客有一定互动功能



##### 核心需求：

1. 文章发布/编辑:支持富文本/Markdown 排版，可插入图片、代码、链接，能保存草稿、定时发布
2. 内容展示:博客首页、文章详情页，按分类、标签、发布时间归档检索
3. 基础导航:首页、分类、标签、关于我、搜索栏等核心入口
4. 个人主页:展示博主简介、头像、联系方式、博客介绍等信息
5. 文章管理:博主后台可编辑、删除、置顶、隐藏已发布文章
6. 评论 / 回复：访客可对文章评论，博主后台审核、回复评论，支持匿名 / 昵称评论
